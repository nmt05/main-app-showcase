Train both hands. Keep both shapes alive.

GameTwoShapes is a divided-attention challenge where players draw and react on two lanes at the same time. The goal is simple: understand the loop fast, see real visuals, and jump into the project quickly.

How The Loop Works

A clear three-step rhythm keeps the game easy to start and difficult to dominate.

01

Read Targets

Two shape targets appear on separate sides. You have limited time to process both at once.

02

Draw Both

Use multi-touch to draw the correct shape on each side. Missing either side breaks the streak.

03

Survive Pressure

Each correct turn boosts score and shortens reaction windows, forcing focus and control.
